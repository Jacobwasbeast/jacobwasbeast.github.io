Configuration

Before using the AOT Minecraft Mod, you may need to configure certain settings:

  • Open the config.json file located in the mod's directory.

  • Customize the mod settings according to your preferences.

  • Save the configuration file.

Example Configuration:

{ "titan_spawn_rate": 0.5, "odm_gear_durability": 500, "custom_items_enabled": true }
