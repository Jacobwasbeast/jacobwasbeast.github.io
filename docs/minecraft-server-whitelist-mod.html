Minecraft Server Whitelist Mod

A Minecraft mod made for servers & clients to only allow progression when a certain amount of players is on the server.

Features:

  • Whitelist functionality to control server access.

  • Customizable settings for whitelist management.

  • Improved server security and management.

Installation:

  1. Download the mod file from here .

  2. Extract the contents into your Minecraft mods folder.

  3. Configure whitelist settings as per your server requirements.

  4. Launch your Minecraft server and enjoy the enhanced control over player access!
