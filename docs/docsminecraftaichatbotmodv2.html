Configuration

Before using the Minecraft AI Chatbot Mod v2, you need to configure it:

  • Open the config.json file located in the mod's directory.

  • Customize the chatbot settings as desired.

  • Save the configuration file.

Example Configuration:

{ "chatbot_name": "AI_Chatbot", "language": "en", "personality": "friendly", "chat_log_file": "chat_logs.txt", "model": "openai" "modelToken": "openai-api-token" }
