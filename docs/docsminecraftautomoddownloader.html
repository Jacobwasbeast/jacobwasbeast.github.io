Mod Usage

1. Configuration

Before using the tool, you need to configure it:

  • Open the config.json file in a text editor.

  • Enter your FTP server details (hostname, port, username, password).

  • Save the configuration file.

Example Configuration:

{ "ftp_server": "ftp.example.com", "ftp_port": 21, "ftp_username": "your_username", "ftp_password": "your_password", "download_directory": "mods", "max_concurrent_downloads": 3 }
